BlogStack Icon Generator

Instructions

This page will help you generate the necessary icon files for your BlogStack PWA:

  1. Click the "Generate Icons" button below
  2. Right-click on each canvas and select "Save Image As..."
  3. Save the 192x192 image as "logo192.png" in this directory
  4. Save the 512x512 image as "logo512.png" in this directory
  5. Save the 64x64 image as "favicon.ico" in the public directory
